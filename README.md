# SpendWise - AI-Assisted Expense Tracker

SpendWise is a modern expense tracking and budgeting platform designed for Indian students and young professionals.

## Features

* Expense Management (Add, Edit, Delete)
* Monthly Budget Tracking
* Savings Goal Tracking
* Spending Analytics
* Category-wise Expense Breakdown
* AI-Assisted Financial Insights
* CSV Export
* Responsive Dashboard

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript
* Chart.js

### Backend

* Python
* Flask

### Database

* SQLite
